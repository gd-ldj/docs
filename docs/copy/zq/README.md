# 早起

转自： [能控制早晨的人，方可控制人生！]()

**快乐感更强**<br />加拿大多伦多大学一项新研究发现，无论是年轻人还是老年人，早起者的情绪更积极，自我健康感更好。早起者沐浴晨光可能是心情舒畅的一大原因。

**成绩更好**<br />美国“网络医学博士”网站刊登得克萨斯大学一项研究发现，“百灵鸟型”(早起)大学生的成绩点比“夜猫子型”(熬夜)大学生整整高出一个点。研究负责人丹尼尔.泰勒博士说，“百灵鸟型”学生生活更规律，准时上课，学习更主动，且没有熬夜带来的健康问题。

**精力更旺盛**<br />研究发现，早起者通常醒得更快，而且头脑更清醒，能够快速投入注意力要求较高的活动，也不容易犯困，精力更旺盛。

**上午效率高**<br />“夜猫子”睡眼惺忪的时候，早起者可能已经完成晨跑、做好早餐、处理完一大堆事务。对于当天更具挑战的活动，早起者更可能会早早制订计划并及时付诸实施。

**做事更认真**<br />多项性格研究发现，责任心强的人效率更高、组织性更强、目标更明确、更关注细节。一些研究发现，早起者头脑更清醒，遇事提前计划，行动更加果敢。

**抑郁风险小**<br />研究发现，爱熬夜的人更容易罹患抑郁症。习惯早起的人做事更井井有条，准备充分，极少慌乱，因而更少抑郁。另一大原因是，早起者睡眠充足，缺少睡眠也是抑郁症的一大诱因。

**为人更和善**<br />美国宾夕法尼亚大学一项研究发现，每晚只睡4.5小时会明显增加暴躁、易怒、悲伤和情绪失控等的几率，更容易与他人发生争吵和误解。早睡早起，这些问题都会得到明显改善。而悉尼大学一项心理学研究发现，习惯早起的人通常更和善。
